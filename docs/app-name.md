## コンセプト

**Continue** = 活動を続けること、継続すること

キャラクターの成長を通じて、日々の活動を継続するモチベーションを提供するアプリ。

### 2つの利用シーン

**🏃 ダイエット目的**
- デブからスリムへの変身を楽しむ
- 有酸素運動や軽めの活動がメイン
- 「痩せていく自分」をキャラで可視化

**💪 筋トレ目的**
- がりがりからムキムキへの成長を楽しむ
- 筋トレ記録がメイン
- 「強くなる自分」をキャラで可視化

どちらの目的でも、同じ活動記録機能を使用。違いはキャラの見た目のみ。
ユーザーは初回起動時に目標を選び、設定でいつでも変更可能。

### キャラクター設計方針

- **全10段階（stage 0-9）で明確な変化**
- 各ステージごとに見た目が変わる
- ダイエットモード: デブ→標準→スリム
- 筋トレモード: がりがり→標準→ムキムキ
- 変化の演出: 0.3〜0.6秒のクロスフェード
