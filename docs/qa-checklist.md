# QA Checklist (per milestone)

## General
- [ ] 起動〜操作でクラッシュしない
- [ ] 画面遷移が崩れない
- [ ] DB保存が効いている（再起動後も残る）

## Stage rules
- [ ] 活動ログが1件でもある日：+1
- [ ] 活動ログが0件の日：-1
- [ ] 休息日は0件でも維持（下がらない）
- [ ] stageは0..9でクリップされる
- [ ] 過去日の追加/削除/休息変更後に、当日以降が再計算される

## Log
- [ ] Strength: 種目追加、セット追加/削除、保存
- [ ] Cardio: minutes保存
- [ ] Light: minutesまたはラベル保存
- [ ] 削除が反映される

## History
- [ ] 直近30日が見える
- [ ] 表示がステージと矛盾しない
