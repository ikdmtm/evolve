# Assets

このディレクトリにはアプリのアセットファイルが含まれています。

## 現在使用中のアイコン

- `icon.png` - アプリアイコン（テキストなし、変身デザイン）
- `splash.png` - スプラッシュ画面（3段階進化、青→オレンジグラデーション）
- `adaptive-icon.png` - Androidアダプティブアイコン（かわいい進化デザイン）
- `favicon.png` - Webファビコン（進化の3段階、円形デザイン）

## 過去バージョン（バックアップ）

- `icon-v1-with-text.png` - 初期デザイン（FitMorphテキスト付き）
- `splash-v1-original.png` - 初期スプラッシュ（流れるような動的デザイン）
- `favicon-v1-original.png` - 初期ファビコン（FMレターデザイン）

## デザインコンセプト

キャラクターの変身・進化をモチーフに、青→紫→オレンジのグラデーションカラーを使用。
シンプルで親しみやすいデザインを重視。
