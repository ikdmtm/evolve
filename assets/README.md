# Assets

このディレクトリにはアプリのアセットファイルが含まれています。

## 現在使用中のアイコン

- `icon.png` - アプリアイコン（テキストなし、変身デザイン）
- `splash.png` - スプラッシュ画面（3段階進化、青→オレンジグラデーション）
- `adaptive-icon.png` - Androidアダプティブアイコン（かわいい進化デザイン）
- `favicon.png` - Webファビコン（進化の3段階、円形デザイン）

## 過去バージョン（バックアップ）

- `icon-v1-with-text.png` - 初期デザイン（FitMorphテキスト付き）
- `splash-v1-original.png` - 初期スプラッシュ（流れるような動的デザイン）
- `favicon-v1-original.png` - 初期ファビコン（FMレターデザイン）

## レベル別キャラクター（ベース画像）

Level 0 のベース画像。ここから Level 1〜10 の変化画像を生成する想定。

| ファイル | 用途 | ベースの体型 |
|----------|------|--------------|
| `character-diet-male-l0-base.png` | ダイエット目的・男性 | 極端に太った状態（→ Level 10 でスリム） |
| `character-diet-female-l0-base.png` | ダイエット目的・女性 | 極端に太った状態（→ Level 10 でスリム） |
| `character-muscle-male-l0-base.png` | 筋トレ目的・男性 | 極端に痩せた状態（→ Level 10 でムキムキ） |
| `character-muscle-female-l0-base.png` | 筋トレ目的・女性 | 極端に痩せた状態（→ Level 10 で筋肉質） |

- いずれも全身・正面・手に何も持たない統一ポーズ。
- レベル進行用に「ひとつ前のレベル画像を入力にして次レベルを生成」する前提でデザイン。

## デザインコンセプト

キャラクターの変身・進化をモチーフに、青→紫→オレンジのグラデーションカラーを使用。
シンプルで親しみやすいデザインを重視。
